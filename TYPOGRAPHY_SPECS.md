# Especificações de Tipografia - CAPIAU

## Logo "CAPIAU" (Hero Section - Matrix Background)

### Fonte
- **Família**: `monospace` (fonte monoespaçada do sistema)
- **Peso**: `bold` (quando completamente formado)
- **Tamanho base**: `17px` (no canvas)
- **Estilo**: Monoespaçada, bold quando formado

### Características Visuais
- **Cor**: `#000000` (preto sólido)
- **Glow Effect**: 
  - Shadow color: `rgba(0, 0, 0, 0.8)`
  - Shadow blur: `40px`
  - Composite operation: `screen` (para o glow)
- **Formato**: Renderizado em canvas HTML5

### Contexto de Uso
O "CAPIAU" é formado dinamicamente no canvas através de uma animação onde os caracteres do matrix se transformam gradualmente na palavra. Quando completamente formado:
- Usa fonte `bold 17px monospace`
- Texto sólido preto com glow effect
- Posicionado centralmente no canvas

## Tipografia do Site

### Fonte Principal: Montserrat
- **Pesos utilizados**: 400, 800, 900
- **Uso**:
  - **400 (normal)**: Textos de corpo, descrições
  - **800 (extra bold)**: Subtítulos, textos de destaque
  - **900 (black)**: Títulos principais, email de contato

### Fonte Secundária: Inter
- **Peso**: Padrão (400)
- **Uso**: Textos auxiliares, UI elements

### Cores
- **Primária**: `#5BBDB4` (turquesa)
- **Secundária**: `#000000` (preto)
- **Texto**: Preto sobre fundo turquesa

## Para Criação de Imagens OG e Materiais

### Recomendações para Imagem Open Graph (1200x630px)

**Texto Principal:**
- "CAPIAU"
- Fonte: Monospace bold
- Tamanho: Ajustar proporcionalmente (sugestão: ~120-150px)
- Cor: Preto (#000000) ou Branco (#FFFFFF) dependendo do fundo
- Glow effect opcional para manter consistência visual

**Subtítulo:**
- "The results-driven, social-first agency you've been looking for."
- Fonte: Montserrat 800 ou 900
- Tamanho: ~40-50px
- Cor: Preto ou branco (contraste com fundo)

**Fundo:**
- Cor primária: `#5BBDB4` (turquesa)
- Ou gradiente turquesa/preto
- Manter minimalista e elegante

### Elementos Visuais Sugeridos
- Logo "CAPIAU" centralizado ou no topo
- Subtítulo abaixo ou sobreposto
- Fundo turquesa (#5BBDB4) com possível gradiente
- Espaçamento generoso para legibilidade
- Manter identidade visual minimalista e premium
